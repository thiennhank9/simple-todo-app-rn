Simple Todo app is written by bare minimum React Native app and Expo.

# Requisites

- Setting up environment for React Native v0.73:
  https://reactnative.dev/docs/environment-setup
  https://docs.expo.dev/

# Run

1. Install dependencies:

```
npm i
```

2. Running for android:

```
npm run android
```

Or for ios:

```
npm run ios
```

Running tests:

```
npm run test
```
